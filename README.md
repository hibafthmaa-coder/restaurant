# 🍽️ Restaurant Website

A simple and responsive restaurant website built using HTML and CSS. This project is designed to practice web development fundamentals and create a clean, attractive landing page for a restaurant.

## ✨ Features

- Responsive layout
- Modern and clean design
- Navigation bar
- Hero section
- Attractive styling with CSS
- Beginner-friendly project structure

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📁 Project Structure

```
restaurant/
│── index.html
│── style.css
└── images/
```

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/hibafthmaa-coder/restaurant.git
   ```

2. Open the project folder.

3. Open `index.html` in your web browser.
